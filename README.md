# Kiev airport clone

## How to start

1. Run npm install at the root folder
2. Start: npm run start
3. Build: npm run build


## Tech stack

1. styled-component
2. Typescript
3. fetch
4. redux-toolkit
5. react, hooks
6. i18next
7. moment, date-fns
8. react-datepicker
9. react-router-dom
